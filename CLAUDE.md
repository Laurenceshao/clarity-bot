# ClarityBot

Slack bot that detects ambiguous/vague messages and replies in-thread with a clarifying question.

## Run
```
cd ~/clarity-bot
.venv/bin/python bot.py
```

## Stack
- Python, slack-bolt (Socket Mode), OpenAI GPT-4o-mini, python-dotenv
